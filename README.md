Sistema de Estoque
Este é um sistema simples de controle de estoque desenvolvido em Python. Ele não possui interface gráfica , sendo operado através do prompt de comando.

Funcionalidades
Login do Usuário: O sistema possui um login de usuário para controlar o acesso ao sistema. O usuário e senha são configuráveis.
Registrar Entrada: Permite ao usuário registrar a entrada de novos produtos no estoque.
Consultar Estoque: Exibe o estoque completo ou permite consultar um produto específico.
Registrar Saída: (Em desenvolvimento) Permite registrar a saída de produtos do estoque.
Ajustar Estoque: (Em desenvolvimento) Permite fazer ajustes nas quantidades de produtos.
Exibir Histórico: (Em desenvolvimento) Exibe o histórico de transações de entrada e saída de produtos.
Tecnologias Usadas
Python 3.x
Como Rodar
Este sistema é operado pelo prompt de comando. Não há interface gráfica
